# Rapido_Cab_Booking_Analysis_Excel_Project
# 🚖 Rapido Cab Booking Analysis Dashboard

## 📌 Project Overview

This project is an interactive Excel dashboard built using Rapido cab booking data to analyze booking performance, revenue trends, customer behavior, and operational efficiency.

The dashboard helps in understanding key business insights such as total bookings, cancellation trends, vehicle performance, customer ratings, and location-wise demand.

---

## 🔧 Tools & Technologies Used

* Microsoft Excel
* Power Query
* Pivot Tables
* Pivot Charts
* Slicers

---

## 📊 Key KPIs

* Total Revenue
* Total Bookings
* Total Customers
* Total Drivers
* Cancellation Percentage
* Average Customer Rating
* Average Ride Distance

---

## 📈 Dashboard Features

* Interactive slicers for dynamic filtering
* Revenue trend analysis
* Hourly booking analysis
* Vehicle-wise booking and revenue analysis
* Location-wise booking insights
* Customer rating analysis
* Booking status analysis

---

## 📌 Key Insights

* Bike rides generated the highest revenue and bookings.
* Delhi recorded the maximum number of bookings.
* Most rides were successfully completed.
* Customer ratings were mostly between 4.5–5.0.
* Peak bookings were observed during late-night and morning hours.

---

## 🎯 Objective

The main objective of this project was to practice data cleaning, data transformation, dashboard creation, and business insight generation using Excel.
